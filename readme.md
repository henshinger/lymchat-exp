## lymchat

### Usage

#### Install Exponent
    [XDE and mobile client](https://docs.getexponent.com/versions/v10.0.0/introduction/installation.html).

#### Install [Lein](http://leiningen.org/#install)

#### Install npm modules

``` shell
npm install
```

#### Start the figwheel server
``` shell
    lein figwheel

```
